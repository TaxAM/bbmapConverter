## How to use it

```
python mappingConverter.py <mapping_path> <output_path> <mappingSeparator> <outputSeparator>
```
For example:
```
python mapping_converter.py /tmp/mapping_01.txt /tmp/mapping_01_converted.txt '\t' '\t'
```
